# Obsidian AI Agent

An AI-powered chat agent plugin for [Obsidian](https://obsidian.md) that lets you communicate with LLM models directly from your vault.

## Features

- Chat with AI models from within Obsidian
- Configurable API endpoint and model selection
- Markdown-only activation policy
- Settings tab for API key, URL, and model configuration

## Development

```bash
pnpm install
pnpm dev    # Watch mode
pnpm build  # Production build
pnpm lint   # Run ESLint
```
